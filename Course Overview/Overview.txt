Welcome to the course Data Visualization for AI and Machine Learning!

In this course we will explore what does it mean to communicate with data and how to create visuals to support your message. We will also learn how to create beautiful, functional and reusable charts using a variety of Python tools. We will have plenty of practical exercises that will accompany the theory sessions.  The primary language of the class is English. 

There is no exam and the course is graded based on two projects: a small intermediate project and a final project.

Semester Outline:

Introduction to data visualization for data science, AI and ML
Visual encodings, human cognition
Chart types, choosing charts, developing custom visualizations
Styling, annotations, elements of graphical design
Colors
Strategies for dealing with complexity
Introduction to interactive visualizations
Survey of Python tools
Introduction to data visualization on the web
Dimensionality reduction for data visualization
Guest lecture
Project work, coaching
Projects - presentations and peer feedback session
